# 🐱 PurrDrop

Cute P2P File Transfer

## ✨ Features

- **WebRTC P2P** - ไฟล์ส่งตรงระหว่าง Browser ไม่จำกัดขนาด!
- **Cute Characters** - สัตว์น่ารักตาม OS
- **Pastel Theme** - UI สีพาสเทลอบอุ่น พร้อม animations
- **Real-time** - เห็นเพื่อนทันที
- **Drag & Drop** - ลากไฟล์วางได้เลย

## 🚀 Quick Start

```bash
npm install
npm run dev
```

เปิด http://localhost:3000

## 📱 How to Use

1. เปิด URL ในอุปกรณ์ที่ 1
2. เปิด URL เดียวกันในอุปกรณ์ที่ 2
3. คลิกที่ตัวละครของเพื่อน → เลือกไฟล์
4. อีกเครื่องกด "รับเลย!"
